---

### Use meaningful and pronounceable variable names

#### Bad:

<Code>const yyyymmdstr = moment().format("YYYY/MM/DD");</Code>

#### Good:

<Code>const currentDate = moment().format("YYYY/MM/DD");</Code>

---

### Use the same vocabulary for the same type of variable

#### Bad:

<Code>
{`getUserInfo();
getClientData();
getCustomerRecord();`}
</Code>

#### Good:

<Code>
{`getUser();`}
</Code>
---

### Use searchable names

##### By not naming variables in a meaningful way, we hurt our readers.

#### Bad:

<Code>
setTimeout(blastOff, 86400000);
</Code>

#### Good:

<Code>
  {`const MILLISECONDS_IN_A_DAY = 86_400_000;
setTimeout(blastOff, MILLISECONDS_IN_A_DAY);`}
</Code>

---

### Avoid Mental Mapping

##### Explicit variables is better than implicit.

#### Bad:

<Code>
  {`const locations = ["Austin", "New York", "San Francisco"];
locations.forEach(l => {
  doStuff();
  doSomeOtherStuff();
  // ...
  // ...
  // ...
  // Wait, what is 'l' for again?
  dispatch(l);
});`}
  </Code>

---

### Avoid Mental Mapping

##### Explicit variables is better than implicit.

#### Good:

<Code>
{`const locations = ["Austin", "New York", "San Francisco"];
locations.forEach(location => {
  doStuff();
  doSomeOtherStuff();
  // ...
  // ...
  // ...
  dispatch(location);
});`}
</Code>

---

### Don't add unneeded context

##### If your class/object name tells you something, don't repeat that in your variable name.

#### Bad:

<Code>
{`const Car = {
  carMake: "Honda",
  carModel: "Accord",
  carColor: "Blue"
};
function paintCar(car) {
  car.carColor = "Red";
}`}
</Code>

---

### Don't add unneeded context

##### If your class/object name tells you something, don't repeat that in your variable name.

#### Good:

<Code>
{`const Car = {
  make: "Honda",
  model: "Accord",
  color: "Blue"
};
function paintCar(car) {
  car.color = "Red";
}`}
</Code>

---

### Use default arguments

#### Bad:
<Code>{`function createMotorcycleBrand(name) {
  const brandName = name || "Harley Davidson";
  // ...
}`}</Code>

#### Too Bad:
<Code>{`function createMotorcycleBrand(name) {
  if (!name)
    const brandName = "Harley Davidson";  
  // ...
}`}</Code>

---

### Use default arguments

#### Really Really Bad:
<Code>{`function createMotorcycleBrand(name) {
  if (name) {
    const brandName = name;  
  } else {
    const brandName = "Harley Davidson";
  }
  // ...
}`}</Code>

---

### Use default arguments

#### Good:
<Code>{`function createMotorcycleBrand(name = "Harley Davidson") {
  // ...
}`}</Code>
